== By Days

Room Booking Application.

Living demo: [http://booking-system-abg.herokuapp.com/](http://booking-system-abg.herokuapp.com/)

Models:

* Hotel
  * id
  * name
  * address
  * description

* Room Type
  * id
  * hotel_id
  * name
  * price

* Room
  * id
  * room_type

* Booking
  * id
  * room_id
  * start_date
  * end_date
  * total_price
